## Production Notes

### Universal Guidelines (All 3 Videos)

**Visual Style:**
- Portrait orientation (9:16 aspect ratio)
- Clean, minimal UI overlays
- Use device frames for professionalism
- Show real hands/faces (builds authenticity)

**Text Overlays:**
- Large, readable sans-serif font (SF Pro or similar)
- High contrast (white text on dark app footage works well)
- 1-2 seconds minimum display time per text
- Align text to top or bottom to avoid blocking important visuals

**Audio Considerations:**
- Silent version (default) - rely on text overlays
- Optional voiceover version for ads/social media
- Keep app's ambient sounds subtle in background
- No music that could conflict with app's calm vibe

**Legal/Compliance:**
- Show app icon and name clearly
- Display price prominently
- Include "Parent Gate protected" or similar security indicator
- No medical/developmental claims without disclaimers

### Post-Production Checklist

- [ ] Color grade for warmth and calm (avoid cold blues)
- [ ] Ensure all text is readable on mobile devices
- [ ] Add subtle fade transitions between scenes (300-500ms)
- [ ] Verify aspect ratio matches App Store requirements
- [ ] Export in highest quality H.264 format
- [ ] File size under 500MB per video
- [ ] Test on actual device before submission

### App Store Specifications

**Technical Requirements:**
- Resolution: 1920x1080 or 1080x1920 (portrait recommended)
- Format: MP4 or MOV
- Codec: H.264
- Frame rate: 30fps minimum
- Duration: 15-30 seconds each
- File size: Max 500MB per video

**Submission Strategy:**
- Upload all 3 videos as preview options
- Order: Video 2 (12-24mo) â†’ Video 1 (6-12mo) â†’ Video 3 (24-36mo)
- Let parents self-select based on child's age
- Consider A/B testing which video performs best as primary

### Alternative Approaches

**If filming babies/toddlers is challenging:**
- Use screen recording only with text overlays emphasizing benefits
- Create animated text cards explaining features
- Use stock footage of babies/toddlers (ensure licensing)
- Focus on app UI with strong messaging

**For social media promotion:**
- Create 10-second cuts of each video for Instagram Stories/Reels
- Add trending audio tracks (replace app sounds)
- Use more dynamic cuts and faster pacing
- Include parent testimonials as text overlays
